= DisableSnmp
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 `DisableSnmp` 方法在集群节点上禁用 SNMP 。



== 参数

此方法没有输入参数。



== 返回值

此方法没有返回值。



== 请求示例

此方法的请求类似于以下示例：

[listing]
----
{
  "method": "DisableSnmp",
  "params": {},
  "id" : 1
}
----


== 响应示例

此方法返回类似于以下示例的响应：

[listing]
----
{
  "result" : {},
  "id" : 1
}
----
