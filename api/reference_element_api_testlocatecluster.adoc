= TestLocateCluster
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 `TestLocateCluster` 方法验证节点是否可以找到在集群配置中指定的集群。输出将验证是否已创建集群，并列出集群集合中的节点。



== Parameters

此方法没有输入参数。



== 返回值

此方法具有以下返回值：

|===
| Name | Description | Type 


 a| 
详细信息
 a| 
有关测试操作成功或失败的信息。
 a| 
JSON 对象

|===


== 请求示例

此方法的请求类似于以下示例：

[listing]
----
{
   "method": "TestLocateCluster",
   "params": {},
   "id" : 1
}
----
