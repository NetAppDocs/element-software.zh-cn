= 使用 Element API 管理存储
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 Element 软件 API 管理 Element 存储集群。

Element API 基于 HTTPS 上的 JSON-RPC 协议。JSON-RPC 是一种基于轻型 JSON 数据交换格式的基于文本的简单 RPC 协议。客户端库可用于所有主要编程语言。
