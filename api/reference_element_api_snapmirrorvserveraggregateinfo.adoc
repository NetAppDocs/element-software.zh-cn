snapMirrorVserverAggregateInfo 对象包含有关目标 ONTAP 集群上可用数据 Storage Virtual Machine （也称为 Vserver ）的信息。



== 对象成员

此对象包含以下成员：

|===
| Name | Description | Type 


 a| 
aggrName
 a| 
分配给 SVM 的聚合的名称。
 a| 
string



 a| 
聚合可用大小
 a| 
分配的聚合的可用大小。
 a| 
整型

|===
