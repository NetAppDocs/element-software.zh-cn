= 集群 API 方法
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以通过 Element 软件集群 API 方法管理存储集群以及属于存储集群的节点的配置和拓扑。

某些集群 API 方法可在属于集群一部分或已配置为加入集群的节点上运行。您可以向新集群或现有集群添加节点。已准备好添加到集群的节点处于 " 待定 " 状态，这意味着它们已配置，但尚未添加到集群。
