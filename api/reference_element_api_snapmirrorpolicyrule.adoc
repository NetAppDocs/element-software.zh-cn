= snapMirrorPolicyRule
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
snapMirrorPolicyRule 对象包含有关 SnapMirror 策略中的规则的信息。



== 对象成员

此对象包含以下成员：

|===
| Name | Description | Type 


 a| 
snapMirrorLabel
 a| 
Snapshot 副本标签，用于在扩展数据保护关系中选择 Snapshot 副本。
 a| 
string



 a| 
keepcount
 a| 
指定规则中 SnapMirror 目标卷上保留的最大 Snapshot 副本数。
 a| 
整型

|===
