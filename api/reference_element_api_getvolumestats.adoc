= GetVolumeStats
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 `GetVolumeStats` 方法获取单个卷的高级别活动测量结果。值是在创建卷时累积的。



== 参数

此方法具有以下输入参数：

|===
| Name | Description | Type | 默认值 | Required 


 a| 
volumeId
 a| 
指定为其收集统计信息的卷。
 a| 
整型
 a| 
无
 a| 
是的。

|===


== 返回值

此方法具有以下返回值：

|===


| Name | Description | Type 


 a| 
volumeStats
 a| 
卷活动信息。
 a| 
xref:reference_element_api_volumestats.adoc[volumeStats]

|===


== 请求示例

此方法的请求类似于以下示例：

[listing]
----
{
   "method": "GetVolumeStats",
   "params": {
      "volumeID": 32
   },
   "id": 1
}
----
