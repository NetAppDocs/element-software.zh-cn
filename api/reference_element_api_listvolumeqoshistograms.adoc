= ListVolumeQoSHistograms
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 `ListVolumeQoSHistograms` 方法生成一个或多个卷的卷 QoS 使用情况直方图。这样，您就可以更好地了解卷如何使用 QoS 。



== Parameters

此方法具有以下输入参数：

|===
| Name | Description | Type | 默认值 | Required 


 a| 
volumeIds
 a| 
一个可选的卷 ID 列表，用于指定应生成 QoS 直方图的卷。
 a| 
整型数组
 a| 
无
 a| 
否

|===


== 返回值

此方法具有以下返回值：

|===


| Name | Description | Type 


 a| 
qosHistogram
 a| 
描述一个或多个卷的卷使用情况的对象列表。
 a| 
JSON 对象数组

|===


== 请求示例

此方法的请求类似于以下示例：

[listing]
----
{
	"method": "ListVolumeQoSHistograms",
		"params": {
			"volumeIDs": [1]
		},
	"id": 1
}
----
