= 提供反馈
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用反馈表帮助改进 Element 软件 Web 用户界面并解决任何 UI 问题，此反馈表可通过整个 UI 访问。

. 在 Element UI 的任何页面中，单击 * 反馈 * 按钮。
. 在摘要和问题描述字段中输入相关信息。
. 附加任何有用的屏幕截图。
. 输入名称和电子邮件地址。
. 选中此复选框可包含有关当前环境的数据。
. 单击 * 提交 * 。
