= 启用 FIPS 驱动器功能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 NetApp Element 软件 `EnableFeature` API 方法启用 FIPS 驱动器功能。

必须在集群上启用空闲加密，并且所有节点和驱动器都必须支持 FIPS ，如 GetFipsReport 为所有节点显示 Ready 状态时所示。

.步骤
. 使用 Element API 输入以下命令，在所有驱动器上启用 FIPS ：
+
`EnableFeature 参数： FipsDrives`
