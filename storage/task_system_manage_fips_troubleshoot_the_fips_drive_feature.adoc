= 对 FIPS 驱动器功能进行故障排除
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用 NetApp Element 软件 UI ，您可以查看有关系统中与 FIPS 驱动器功能相关的集群故障或错误信息的警报。

. 使用 Element UI ，选择 * 报告 * > * 警报 * 。
. 查找集群故障，包括：
+
** FIPS 驱动器不匹配
** FIPS 驱动器不合规


. 有关解决方案建议，请参见集群故障代码信息。
