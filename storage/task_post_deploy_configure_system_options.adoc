= 在部署后配置 SolidFire 系统选项
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
设置 SolidFire 系统后，您可能需要执行一些可选任务。

如果更改系统中的凭据，您可能希望了解对其他组件的影响。

此外，您还可以配置多因素身份验证，外部密钥管理和联邦信息处理标准（ FIPS ）安全性的设置。您还应根据需要查看更新密码。
