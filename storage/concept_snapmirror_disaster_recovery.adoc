= 使用 SnapMirror 进行灾难恢复
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果运行 NetApp Element 软件的卷或集群出现问题，请使用 SnapMirror 功能中断关系并故障转移到目标卷。


NOTE: 如果原始集群完全出现故障或不存在，请联系 NetApp 支持部门以获得进一步帮助。
