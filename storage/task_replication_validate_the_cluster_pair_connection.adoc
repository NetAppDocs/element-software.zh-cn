= 验证集群对连接
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
完成集群配对后，您可能需要验证集群对连接以确保复制成功。

. 在本地集群上，选择 * 数据保护 * > * 集群对 * 。
. 在 * 集群对 * 窗口中，验证集群对是否已连接。
. * 可选： * 导航回本地集群和 * 集群对 * 窗口，并验证集群对是否已连接。
