= 复制后删除卷关系
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
复制完成后，如果您不再需要此卷对关系，则可以删除此卷关系。

. 选择 * 数据保护 * > * 卷对 * 。
. 单击要删除的卷对对应的 * 操作 * 图标。
. 单击 * 删除 * 。
. 确认消息。
