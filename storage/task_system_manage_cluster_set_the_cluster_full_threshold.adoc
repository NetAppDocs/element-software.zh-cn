= 设置集群全满阈值
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用以下步骤更改系统生成块集群填充度警告的级别。此外，您还可以使用 ModifyClusterFullThreshold API 方法更改系统生成块或元数据警告的级别。

.您需要的内容
您必须具有集群管理员权限。

.步骤
. 单击 * 集群 * > * 设置 * 。
. 在 Cluster Full Settings 部分的 * 当 Helix 无法从节点故障中恢复之前剩余 _% 容量时发出警告警报 * 中输入一个百分比。
. 单击 * 保存更改 * 。
