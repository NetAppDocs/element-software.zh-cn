= 编辑组快照
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以编辑现有组快照的复制和保留设置。

. 单击 * 数据保护 * > * 组快照 * 。
. 单击要编辑的组快照对应的 "Actions" 图标。
. 在显示的菜单中，选择 * 编辑 * 。
. * 可选： * 要更改组快照的复制设置，请执行以下操作：
+
.. 单击 * 当前复制 * 旁边的 * 编辑 * 。
.. 选中 * 配对时将每个组快照成员包括在复制中 * 复选框，以确保配对父卷时在复制中捕获每个快照。


. * 可选： * 要更改组快照的保留设置，请从以下选项中进行选择：
+
.. 单击 * 当前保留 * 旁边的 * 编辑 * 。
.. 为组快照选择一个保留选项：
+
*** 单击 * 永久保留 * 可在系统上无限期保留快照。
*** 单击 * 设置保留期限 * ，然后使用日期调整框选择系统保留快照的时间长度。




. 单击 * 保存更改 * 。
