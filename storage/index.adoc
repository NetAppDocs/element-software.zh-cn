= 使用 Element 软件管理存储
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用 Element 软件设置 SolidFire 存储，监控集群容量和性能以及管理多租户基础架构中的存储活动。

Element 是 SolidFire 集群的核心存储操作系统。Element 软件可在集群中的所有节点上独立运行，并可使集群节点将资源组合在一起，并作为一个存储系统提供给外部客户端。Element 软件负责整个系统的所有集群协调，扩展和管理工作。

软件界面基于 Element API 构建。
