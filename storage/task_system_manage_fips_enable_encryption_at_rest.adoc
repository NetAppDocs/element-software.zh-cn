= 启用空闲加密
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以启用和禁用集群范围的空闲加密。默认情况下，不会启用此功能。要支持 FIPS 驱动器，必须启用空闲加密。

. 在 NetApp Element 软件 UI 中，单击 * 集群 * > * 设置 * 。
. 单击 * 启用空闲加密 * 。
