= 管理帐户
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在 SolidFire 存储系统中，客户端可以通过用户帐户连接到节点上的卷。创建卷时，卷会分配给特定的用户帐户。您还可以管理 SolidFire 存储系统的集群管理员帐户。
