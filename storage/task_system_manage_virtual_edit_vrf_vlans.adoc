= 编辑 VRF VLAN
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以更改 VRF VLAN 属性，例如 VLAN 名称，网络掩码，网关和 IP 地址块。

. 单击 * 集群 * > * 网络 * 。
. 单击要编辑的 VLAN 对应的 "Actions" 图标。
. 单击 * 编辑 * 。
. 在 * 编辑 VLAN* 对话框中输入 VRF VLAN 的新属性。
. 单击 * 保存更改 * 。
